# Otne 0.0.0.1 (中引编程语言)

bplm和otne 离不开plug/module关键词的。Otne语法很像Go和java语法这样。

Otne/中引编程语言是2022年3月6日开始仿CPP语法设计改进，一门新的语法编程语言，由 Uxnt 开发。 

[简体中文](README.md) • [English](README.en.md)  


打印42示例代码：
```kt
module stdio;
fn main () {
    printf("42")
}
// fn main print "42";
// func main print "42";
```

"你好，世界" 示例代码：
```kt
module stdio;
fn main () {
    printf("Hello, World !")
}

// fn main print "Hello, World !";
// func main print "Hello, World !";
```
